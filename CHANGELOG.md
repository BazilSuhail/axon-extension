# Changelog

All notable changes to this project will be documented in this file.

## [0.0.1] - 2024-03-21

### Added
- Initial release of API Tester extension
- Support for GET, POST, PUT, PATCH, DELETE, HEAD, and OPTIONS HTTP methods
- Request builder with URL input
- Custom headers support with add/remove functionality
- Request body editor for POST, PUT, and PATCH requests
- Response viewer with status code and formatted output
- JSON syntax highlighting in responses
- Dark and light theme support through VS Code theme integration
- Command: "Open API Tester" to launch the extension panel

### Features
- Lightweight and fast API testing directly in VS Code
- No external dependencies or API keys required
- Works with any REST API
- Built-in JSON formatting for responses
